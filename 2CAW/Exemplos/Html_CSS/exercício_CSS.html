Conhecendo o CSS

Você conhece as formas de uso do CSS? Já ouviu falar de seletores?

Formas de implementação do CSS:

  • CSS inline: aplicado a cada tag dentro do código HTML. É necessário inserir na tag 'style: "parâmetro:valor"
  • CSS interno: aplicado no "head" dentro da tag style. Permite a estilização de todas as ocorrências do seletor indicado na página HTML;
  • CSS externo: link para um arquivo externo que contém informações de estilos. Normalmente usado para layouts e Google Fonts.

Seletores

É possível através da escolha dos seletores definir onde os estilos criados serão aplicados:

  • Seletor tag: ao usar a tag HTML como seletor, definimos que o estilo criado será aplicada em todas as ocorrorências da tag na página;
  • Seletor classe: permite a criação de um conjunto de estilos que pode ser aplicada a tags específicas. Precisa do "." antes do nome no "style" e a chamada na tag usando "class=nome";
  • Seletor ID: semelhante à classe, porém usado para estruturas únicas da página. Usa-se "#" antes do nome em "style" e chama-se na tag com "id=nome".
